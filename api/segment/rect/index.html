    Rect Segment

    Creates a Rectangle made out of 2 triangle/wedges and positioned at a given point in the track.

    RectSegment

    Extends Segment

    RectSegment : Segment {
    Wedge : WedgePart,

    P0 : Vector3,
    P1 : Vector3,
    P2 : Vector3,
    P3 : Vector3,
}

    Properties

    Wedge

    The WedgePart used to make the rectangle. This will be cloned each time a new segment is created.
